# Мастерство приготовления кофе и чая

Образовательный курс для бариста — серия HTML-презентаций (7 модулей) + индекс-файл-навигатор.

## Структура проекта

```
/
├── index.html                          # Главная страница-навигатор
├── brand-spec.md                       # Спецификация дизайн-системы
├── presentations/
│   ├── module-0-coffee-tea-basics.html   # Модуль 0 — Основы кофе и чая (12 слайдов)
│   ├── module-1-cupping.html             # Модуль 1 — Каппинг и органолептика (11 слайдов)
│   ├── module-2-espresso.html            # Модуль 2 — Эспрессо и кофемашина (11 слайдов)
│   ├── module-3-latte-art.html           # Модуль 3 — Латте-арт (11 слайдов)
│   ├── module-4-alternative-brewing.html # Модуль 4 — Альтернативные методы заваривания (11 слайдов)
│   ├── module-5-tea-mastery.html         # Модуль 5 — Мастерство чая (16 слайдов)
│   └── module-6-final-exam.html          # Модуль 6 — Итоговая аттестация (10 слайдов)
└── assets/                              # Директория для дополнительных ресурсов
```

## Как использовать

1. Откройте `index.html` в браузере — это навигатор по курсу.
2. Нажмите на карточку модуля, чтобы открыть соответствующую презентацию.
3. Внутри презентации используйте:
   - **Клавиши ← →** или **пробел** для переключения слайдов
   - **Свайп** на сенсорных экранах
   - **Home / End** для перехода к первому/последнему слайду

## Дизайн-система

- Фон: #1a120e (глубокий тёмно-коричневый)
- Акцент: #d4a574 (золотисто-карамельный)
- Текст: #f5f0eb (тёплый белый)
- Шрифты: Cormorant (заголовки) + DM Sans (основной текст)
- Каждый модуль имеет уникальный цвет-акцент

## Формат

Все файлы — самодостаточные HTML-документы с встроенными стилями и скриптами.
Требуется только браузер — установка дополнительного ПО не нужна.
