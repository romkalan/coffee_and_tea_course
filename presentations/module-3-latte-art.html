Модуль 3

Устройство стимера

Пароблок
Steam wand

Металлическая трубка, подаёт пар из бойлера. Угол наклона влияет на завихрение молока.

Насадка
Tip (1–4 отверстия)

Однодырочная — точный контроль. Многодырочная — быстрый нагрев. Глубина погружения определяет качество пены.

Регулятор пара
Контроль подачи

Рычаг или вентиль. Открывается полностью — давление 1–1.5 бар. Плавное закрытие после вспенивания.

Дренаж продувки
Purge

Продувка пара перед использованием удаляет конденсат. Протирка насадки после — гигиена и долговечность.

Техника

Наклон питчера 20–30°, насадка у поверхности молока. Глубина погружения 1–2 мм на этапе аэрации, затем чуть глубже для воронки.

Модуль 3

Физика вспенивания

Микропузырьки
0.1–0.5 мм

Бархатистая текстура, глянцевый блеск. Пена стабильная, держится до 5 минут. Идеальна для латте-арта.

Крупные пузыри
> 1 мм

Нестабильная пена, быстро оседает. Рисунок расплывается. Результат неправильной аэрации или перегрева.

Состав молока

Жирность 3.2–3.5% — оптимальна: жир обволакивает пузырьки, не даёт им сливаться. Белок казеин создаёт структуру пены. Молоко 1.5% даёт крупные пузыри.

Модуль 3

Типы молочной текстуры

Капучино

Плотная пена, слой 1–1.5 см.
Температура 60–65°C.
Рисунок не обязателен — важна текстура.

Латте

Тонкая пена ~0.5 см.
Более жидкое молоко.
Основа для латте-арта, высокая текучесть.

Флэт-уайт

Микропена без видимого слоя.
Гладкая поверхность.
Австралийский стандарт, самая тонкая текстура.

Модуль 3

Этапы вспенивания

01

Аэрация

2–3 сек. Насадка у поверхности. Шипящий звук — впуск воздуха.

02

Текстурирование

5–10 сек. Насадка погружена. Воронка разбивает пузыри.

03

Нагрев

До 60–65°C. Без аэрации — только нагрев. Питчер горячий — пора выключать.

04

Очистка

Продувка пара. Протирка насадки. Гигиена перед следующим использованием.

Модуль 3

Базовые элементы

● Точка

Базовый элемент. Налив с малой высоты — молоко вытекает, образуя белое пятно на крема.

♡ Сердце

Точка + проход через центр. Молоко растекается симметрично, образует форму сердца.

❀ Тюльпан

3–4 слоя. Последовательные наливы в одну точку. Каждый слой толкает предыдущий.

Модуль 3

Типичные ошибки

!

Пена слишком жидкая

Быстро оседает — недостаточная аэрация. Увеличь время насадки у поверхности до 3–4 сек.

!

Крупные пузыри

Неправильный угол питчера. Держи питчер под наклоном 20–30° для правильного завихрения.

!

Нет контраста

Эспрессо слишком светлый или старая крема. Используй свежий кофе с плотной пенкой.

!

Рисунок расплывается

Слишком жидкое молоко — недостаточно текстурирования. Добейся гладкой микропены.

!

Симметрия нарушена

Кривой налив. Держи питчер ровно, целься в центр чашки с одинаковой высоты.

Модуль 3 · Глоссарий

Глоссарий

  • Микропенка — мелкодисперсная пена с пузырьками < 0.5 мм, гладкая и блестящая
  • Латте-арт — рисунок на поверхности кофе, создаваемый вливанием вспененного молока
  • Питчер — металлический кувшин для вспенивания и налива молока
  • Стимер — пароблок кофемашины для вспенивания молока
  • Аэрация — этап впуска воздуха в молоко через насадку стимера
  • Текстурирование — этап создания завихрения для равномерной микропены
  • Крема — плотная золотистая пенка на поверхности свежесваренного эспрессо
